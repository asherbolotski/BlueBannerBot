Channel Status LED
LED Status
LED Description
When
Component Status
LED off
Anytime
Channel has voltage and is operating as expected
Red Solid
Anytime
Channel has NO voltage and there is an active fault. Check for tripped or missing circuit breaker
Red Blink
Anytime
Sticky fault on the channel. Check for tripped circuit breaker.
Sticky Fault
- an indicator that a fault has occurred that will stay until the indicator has been cleared manually.
