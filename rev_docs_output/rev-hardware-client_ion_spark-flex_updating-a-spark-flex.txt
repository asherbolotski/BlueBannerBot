Updating a Single SPARK Flex
Connect your SPARK Flex Motor Controller to your computer with a USB-C cable.
Open the REV SPARK Flex Client application.
The Client should automatically scan and connect to your SPARK Flex.
Once the SPARK Flex is connected via USB-C select it within the
Connected Hardware.
SPARK Flex needing updates in the RHC
The Hardware Client will open up on the
Basic
tab. To update firmware select the
Update
tab after selecting the SPARK Flex from the Connected Hardware list.
SPARK Flex "Basic" Tab with the "Update" Tab highlighted
Under
SPARK Flex Firmware
, select download to download the latest version of the firmware.
SPARK Flex Download Button
Once the firmware has downloaded select update.
SPARK Flex Update Button
The update process will flash the firmware image onto the SPARK Flex. The status bar will show the progress of the process.
SPARK Flex currently updating
Once the firmware update is done your SPARK Flex will show a new status of
Up-to-Date.
A SPARK Flex shown as Up-to-Date
