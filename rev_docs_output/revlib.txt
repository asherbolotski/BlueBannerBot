Overview
REVLib is a software library that supports various REV devices within the context of programming in the
FIRST
Robotics Competition.
The following devices are supported in REVLib:
SPARK MAX
SPARK Flex
Color Sensor V3
Servo Hub
Use of the library outside of FRC is not currently supported
