Set the output power behavior when the channel is disabled.
When the channel is enabled [ServoChannel.setEnabled(true)], the output power to the servo
 follows the channel's power setting [ServoChannel.setPowered()].
When the channel is disabled [ServoChannel.setEnabled(false)], the output power to the servo
 follows the channel's disableBehavior.

behavior
