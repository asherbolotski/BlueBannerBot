Create a new configuration for a SysId test routine.
Parameters:
rampRate
- The voltage ramp rate used for quasistatic test routines. Defaults to 1 volt
     per second if left null.
stepVoltage
- The step voltage output used for dynamic test routines. Defaults to 7
     volts if left null.
timeout
- Safety timeout for the test routine commands. Defaults to 10 seconds if left
     null.
recordState
- Optional handle for recording test state in a third-party logging
     solution. If provided, the test routine state will be passed to this callback instead of
     logged in WPILog.
