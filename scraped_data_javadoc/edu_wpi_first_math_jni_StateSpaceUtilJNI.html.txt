boolean
isStabilizable
(int states,
 int inputs,
 double[] A,
 double[] B)
Returns true if (A, B) is a stabilizable pair.
(A, B) is stabilizable if and only if the uncontrollable eigenvalues of A, if any, have
 absolute values less than one, where an eigenvalue is uncontrollable if rank(lambda * I - A, B)
 < n where n is the number of states.
Parameters:
states
- the number of states of the system.
inputs
- the number of inputs to the system.
A
- System matrix.
B
- Input matrix.
Returns:
If the system is stabilizable.
