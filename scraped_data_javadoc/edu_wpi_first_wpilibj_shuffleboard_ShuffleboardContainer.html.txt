void
buildInto
(
NetworkTable
parentTable,
NetworkTable
metaTable)
Builds the entries for this value.
Parameters:
parentTable
- the table containing all the data for the parent. Values that require a
     complex entry or table structure should call
parentTable.getSubTable(getTitle())
to
     get the table to put data into. Values that only use a single entry should call
parentTable.getEntry(getTitle())
to get that entry.
metaTable
- the table containing all the metadata for this value and its sub-values
